  • Project Description
    Develop a Sailfish OS port for the Fairphone 4, a task that has not been undertaken previously. Our project group, consisting of four other members, was assigned the seamless integration of Sailfish OS on the Fairphone 4, encompassing all essential functions. An added opportunity presented itself: the ability to contribute our progress to the dedicated open-source communities supporting both Fairphone and Sailfish OS.

    The Fairphone 4
    A modular smartphone, made with responsibly sourced materials and is designed for sustainability and reducign electronic waste.

    Sailfish OS
    A mobile operating system, based on Linux, known for its privacy features, gesture-based navigation, and Android app compatibility.

    Results
    After an intensive five-month development phase, we successfully implemented all essential smartphone functionalities in our custom ROM. We packaged the solution into an accessible image and created a comprehensive installation guide, which was shared on GitHub.

    We concluded the project by sharing our progress through posts on forums like XDA and Faiphone in order to fulfill our commitment to the open-source community; making a valuable contribution to collective knowledge and advancements.

  • Project Description
    Developed an interpreter and compiler for a self-designed Turing Complete programming language, utilizing functional programming in Python, with the objective of conducting unit tests on an Embedded System, specifically the Arduino Due, to validate the language's functionality.

    Python
    Python is a high-level programming language known for its simplicity and readability, widely used for various applications in software development.

    Arduino
    Arduino is an open-source electronics platform based on easy-to-use hardware and software.

    Results
    The final product exists of 4 components:
    "Symbolic" the self-designed Turing Complete programming language.
    The Interpreter emulates "Symbolic" with integrated lexer and parser.
    The Compiler Utilizes the lexer and parser to translate "Symbolic" code into Assembly.
    The Arduino Due Setup consisting of makefiles, the UART driver, timers and essential tools for program flashing and execution.

    All components are openly accessible on GitHub.
